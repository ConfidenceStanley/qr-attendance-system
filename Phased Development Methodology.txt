## GitHub Repository Setup (Do This First)

```
STEP 1: Create GitHub Account if you dont have one
        → github.com/signup

STEP 2: Create New Repository
        → Repository name: qr-attendance-system
        → Description: QR Code Based Attendance 
                        Monitoring System
        → Set to: Public
        → Check: Add README.md
        → Check: Add .gitignore (Choose Node)
        → Click: Create Repository

STEP 3: Create the Folder Structure on GitHub
        The repo will have 3 main folders:
        ┌─────────────────────────────┐
        │   qr-attendance-system/     │
        │   ├── backend/              │
        │   ├── web-frontend/         │
        │   └── mobile-app/           │
        └─────────────────────────────┘

STEP 4: Clone to your computer
        → Open terminal/cmd
        → git clone https://github.com/
              YOUR_USERNAME/qr-attendance-system.git
        → cd qr-attendance-system

STEP 5: Create Branch Strategy
        main        → stable/production code
        develop     → active development
        feature/*   → individual features

        Commands:
        git checkout -b develop
        git push origin develop

STEP 6: Set develop as default working branch
        All phases will be built on develop branch
        Only merge to main when phase is complete
```
